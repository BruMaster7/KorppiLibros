COMPRAS EN MERCADO LIBRE

Korppi es una página web filtra resultados de Mercado Libre para libros, ofreciendo resultados precisos y herramientas de comparación de precios y calidad.

¿Qué es Korppi?

Es un proyecto de web que busca facilitar la búsqueda de Libros por Mercado Libre haciendo uso de su API para Desarrolladores.
